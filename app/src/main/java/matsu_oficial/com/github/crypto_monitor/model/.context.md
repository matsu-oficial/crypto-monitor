Camada de modelo contendo data classes para desserialização JSON das respostas da API Mercado Bitcoin usando Retrofit/Gson.

## Summary

- `TickerResponse.kt` — classes de resposta (TickerResponse e Ticker) mapeando o JSON da API de ticker do Bitcoin

## Constraints

- MUST manter nomes de propriedades sincronizados com JSON da API (high, low, vol, last, buy, sell, date)
- MUST usar tipos apropriados (String para valores monetários, Long para timestamps)

## Guidance

- Adicionar @SerializedName se o JSON da API diferir do nome da propriedade Kotlin
- Considerar usar data classes imutáveis para valores monetários em futuras melhorias
