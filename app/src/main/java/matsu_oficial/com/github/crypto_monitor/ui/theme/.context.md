Configuração centralizada de tema Material 3 para a aplicação, incluindo paleta de cores, tipografia e estilos aplicáveis globalmente.

## Summary

- `Color.kt` — definição de cores do tema (cores primárias, secundárias, etc.)
- `Theme.kt` — implementação do tema Material 3 (CryptomonitorTheme) aplicando cores e tipografia
- `Type.kt` — definição de estilos de tipografia (body, heading, etc.)

## Constraints

- MUST manter compatibilidade com Material 3 API
- MUST usar CryptomonitorTheme em todas as Composables principais

## Guidance

- Atualizar Color.kt para refletir identidade visual da aplicação
- Adicionar mais estilos em Type.kt conforme necessário
- Usar Theme.kt para centralizar lógica de tema (dark/light mode)
