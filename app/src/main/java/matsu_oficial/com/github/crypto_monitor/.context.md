- `MainActivity.kt` — Activity principal que inicializa a aplicação com Jetpack Compose e Scaffold base
- `model/` — classes de dados para serialização/desserialização de respostas da API
- `service/` — interfaces Retrofit para chamadas HTTP
- `ui/theme/` — configuração de cores, tipografia e tema Material 3

## Constraints

- MUST usar Retrofit para todas as chamadas HTTP
- MUST aplicar CryptomonitorTheme em todas as Composables principais

## Guidance

- Expandir MainActivity com mais Composables conforme a UI cresce
- Manter model/ sincronizado com estrutura JSON da API Mercado Bitcoin
- Adicionar ViewModel quando lógica de estado ficar complexa
