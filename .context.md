Projeto de aplicação Android em Kotlin que monitora preços de criptomoedas, especificamente Bitcoin, integrando com a API do Mercado Bitcoin. É um projeto de university/college para estudar Kotlin, Android Compose e integração de APIs REST.

## Summary

- `app/` — módulo principal da aplicação Android com código-fonte e recursos
- `gradle/` — configuração de wrapper do Gradle
- `build.gradle.kts` — arquivo de build do projeto raiz
- `settings.gradle.kts` — configuração de módulos e repositórios
- `gradle.properties` — propriedades globais do Gradle
- `CLAUDE.md` — instruções de contexto para agentes Claude
- `AGENTS.md` — framework de personas de IA para desenvolvimento

## Constraints

- MUST responder em português (BR) conforme `CLAUDE.md`
- MUST ler `AGENTS.md` antes de iniciar qualquer trabalho
- MUST manter `.agents/` intacto (managed via git)

## Guidance

- O projeto usa Jetpack Compose para UI
- Integra Retrofit para chamadas HTTP
- Arquitetura segue padrão Android moderno (minSdk 24, compileSdk 36)
